/pwn_bootcamp

These are all my writeups for the pwn bootcamp run by the Texas A&M Cybersecurity Club. I might add detailed explanations later, but this is mostly a quick reference for copy-pasting into future solve scripts, so it assumes basic knowledge of what you're supposed to be doing.

Week 01

- Flag Hole: buffer overflow overwrite variable
- Login: bof overwrite RIP
- Sally: bof jump to shellcode at leaked stack address

Week 02

- args: ROPchain modify registers and call function
- leaked: use leaked system address to call system('/bin/sh')
- leakme: leak libc using puts and call system('/bin/sh')

Week 03

- minimal: syscall to call execve("bin/sh", 0, 0)
- minimal 2: syscall to mprotect a region of memory to be rwx, then syscall to read shellcode to it and jump to the shellcode

Week 04

- pivot: stack pivoting, the address of user input is leaked and leave;ret instructions are used to get to ropchain
- relative: stack pivoting by using add rsp instructions

Week 05

- echo1: printf leaking stack values
- echo2: printf overwrite variable
- echo3: leaking libc and main, then overwriting the GOT entry for exit to jump to a ROPchain
